# Python Programming

## How to Run

Make sure you have python installed on your system.

Run each task using the following commands:

```bash
python Task1.py

python Task2.py
```

Task 1:
This program takes an integer numbers as a user input and checks whether the number is even or odd and displays the output.

Task 2:
This program calculates the sum of all integers in the range (1 to 50) using for loop and the displays the output.
